    4. SC-4 (running bar + create + editor + delete): Start a timer (Timers tab), return to Gantt → that entry is a live accent/pulsing bar (D-19) whose right edge advances toward "now" (D-13); its right edge is NOT draggable but its left edge is; clicking the stop icon stops the timer and then the end edge becomes draggable.
